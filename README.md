# 🚗 Car Racing Game

A simple and fun **Car Racing Game** built using **HTML, CSS, and JavaScript**.  
Control your car, avoid obstacles, and try to achieve the highest score!

---

## 🎮 Features

- 🕹️ Smooth car controls
- 🚧 Randomly generated obstacles
- 📈 Score tracking system
- ⚡ Fast and lightweight gameplay
- 🎨 Simple and clean UI

---

## 🛠️ Tech Stack

- **HTML** – Structure of the game  
- **CSS** – Styling and layout  
- **JavaScript** – Game logic and interactions
